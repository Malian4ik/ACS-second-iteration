# Avulus: структура сайта и переходы

Короткое описание сайта по страницам. Документ фиксирует, что находится на каждой странице и куда ведут основные ссылки и CTA.

## Общее для всех страниц

- Во всех страницах есть верхняя навигация и общий footer.
- Главная CTA в шапке ведёт в основной целевой сценарий страницы.
- В footer на всех страницах доступны: бронь клуба, Telegram, VK и звонок.

## 1. Главная страница `/`

### Что это за страница

Главная страница делит пользователя на два основных сценария: клуб и ресторан.

### Что на странице есть

- Hero с двумя основными CTA.
- Блок форматов: Stream / Solo, Privat / VIP, Team / Bootcamp.
- Два больших блока: Клуб и Ресторан.
- Контакты и карта внизу страницы.

### Куда ведут ссылки и CTA

- Навигация: Клуб → якорь клубного блока, Ресторан → якорь ресторанного блока, Контакты → блок контактов.
- Кнопка «Забронировать комп» → внешняя бронь клуба.
- Кнопка «Зарезервировать стол» → Telegram.
- Карточки форматов → `/rooms` с выбранным фильтром.
- Кнопка в блоке «Клуб» → внешняя бронь клуба, ссылка «Страница клуба» → `/cyberclub`.
- Кнопка в блоке «Ресторан» → Telegram, ссылка «Страница ресторана» → `/restaurant`.

![Скрин: главная страница](./review-assets/home-full.png)

## 2. Страница клуба `/cyberclub`

### Что это за страница

Страница клуба подробно раскрывает клубный продукт: зоны, тарифы и быстрый переход к брони.

### Что на странице есть

- Hero клуба с основным описанием и быстрыми кнопками.
- Секция зон и форматов клуба.
- Полный тарифный блок.
- Финальный блок брони и звонка.

### Куда ведут ссылки и CTA

- Навигация: Главное → верх страницы, Зоны → секция зон, Тарифы → тарифный блок, Бронь → финальный блок.
- Кнопка «Забронировать комп» → внешняя бронь клуба.
- Кнопка «Фото и железо» → внешняя ссылка на железо/страницу клуба.
- Внизу страницы кнопка брони → внешняя бронь клуба.
- Внизу страницы кнопка с номером → звонок.

![Скрин: страница клуба](./review-assets/cyberclub-full-latest.png)

## 3. Страница ресторана `/restaurant`

### Что это за страница

Страница ресторана показывает ресторан как отдельный продукт с меню и быстрым резервом.

### Что на странице есть

- Hero ресторана с реальными фото.
- Блок позиционирования ресторана.
- Секция меню с категориями и ценами.
- Финальный блок резерва.

### Куда ведут ссылки и CTA

- Навигация: Главное → верх страницы, Меню → секция `menu`, Резерв → финальный блок `reserve`.
- Кнопка «Зарезервировать стол» в шапке и в hero → Telegram.
- Кнопка «Написать в VK» → VK.
- Внизу страницы Telegram → Telegram.
- Внизу страницы номер → звонок.

![Скрин: страница ресторана](./review-assets/restaurant-full-latest.png)

## 4. Страница комнат `/rooms`

### Что это за страница

Страница комнат нужна для выбора конкретного формата с быстрым переходом к брони или контакту.

### Что на странице есть

- Hero с объяснением выбора формата.
- Фильтр: Все комнаты, Solo, Privat / VIP, Bootcamp.
- Карточки комнат с фото, ценой, вместимостью и CTA.
- Блок контактов внизу страницы.

### Куда ведут ссылки и CTA

- Навигация: Главная → `/`, Комнаты → `/rooms`, Контакты → блок `contact-options`.
- Верхняя кнопка «Написать по командам» → Telegram.
- В каждой карточке кнопка брони → внешняя бронь клуба.
- В каждой карточке кнопка «Спросить в VK» → VK.
- В нижнем блоке: бронь → внешняя бронь клуба, Telegram → Telegram, Позвонить → звонок.
